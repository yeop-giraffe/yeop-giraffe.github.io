아래 명령어 중 하나 입력

1
2

$ git config --global user.email "email address"
$ git config --global user.name "user name"

Git push origin main에서 발생

2

$ git remote rm origin  //연결 해제

$ git push origin main  //reject 오류: git push origin +main

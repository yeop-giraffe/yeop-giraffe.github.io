At Samsung Electronics, worked on robotic vacuum cleaner mop module mechanism design.

The work included prototyping a mop attachment and detachment detection mechanism, improving detection performance using a shielding plate, designing structures for module durability, developing performance evaluation methods, supporting VR90F launch readiness, and implementing a vision inspection system for production quality control.
